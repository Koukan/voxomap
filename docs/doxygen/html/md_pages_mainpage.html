Octree Representation

The library is based on an octree, it's a tree data structure in which each internal node has exactly eight children. Octrees are used to partition a three-dimensional space by recursively subdividing it into eight octants/nodes. Octrees are the three-dimensional analog of quadtrees.

Voxel Area Representation

The leaves of the VoxelOctree, that have size of 8, contain VoxelArea. A VoxelArea is structure with a 3D array of voxels of size 8x8x8, so it can contains 512 voxels. The VoxelArea fills all its octants/nodes, so like its container, it is axis aligned.
