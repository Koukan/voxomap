Vector3

Vector3 is a simple class that defines a mathematical vector with three coordinates (x, y and z). It can be used to represent anything that has three dimensions: a size, a point, a velocity, etc.

The Vector3 class has a small and simple interface, its x, y and z members can be accessed directly.

Usage example:

Vector3 v1(16.5, 24, -8.2);
v1.x = 18.2f;
double y = v1.y;
double z = v1.z;
Vector3 v2 = v1 * 5;
Vector3 v3;
v3 = v1 + v2;
std::cout << "v1 = " << v1 << std::endl;
std::cout << "v2 = " << v2 << std::endl;
if (v2 != v3)
std::cout << "v3 = " << v3 << std::endl;

Output:

v1 = 18.2 24 -8.2
v2 = 91 120 -41
v3 = 109.2 144 -49.2
